Čau… planeto?

Ahoj. Jestli tohle čteš, tak jsem pravděpodobně mrtvý… snad ne. Sociálně ale možná dost jo. Tohle je můj první blog post mimo vývojářskou sérii. Když to píšu, docela se bavím sám sebou. Vést blog nebude vůbec jednoduché, protože ani sociální sítě nestíhám zásobovat včas.

Takže prosím nečekej každodenní zábavu na čas. Klidně se stav třeba jednou za měsíc. Tyhle vedlejší zápisy budou víc osobní, ale zároveň se nechci utopit v balastu. Budu se snažit do každého postu přidat alespoň něco užitečného. Dneska mám něco malého, co by se mohlo hodit třeba pro vlastní web.

Něco užitečného?

Možná tě napadá: jak tenhle opičák dostal web online a jak to, že za to skoro nic neplatí? No, já a můj kolega Daniel Hruška jsme našli pár zajímavých řešení hluboko na GitHubu. Jeden chytrý německý webdesigner používá JS compiler, který umí převádět blog posty napsané v markdownu do HTML souborů.

A teď klasická otázka: „Takže máš primitivní backend, ale jak to, že za hosting neplatíš?“ GitHub ti totiž dovolí mít jednu osobní stránku úplně zdarma. Jediné, co si případně musíš zařídit a trochu zaplatit, je vlastní doména. Takže jo, není to úplně stoprocentně zdarma, ale je to sakra levné.

Pokud tě to zajímá víc, můžeš si o tom řešení přečíst tady: Minimal-viable-blog

A protože jsem tehdy ještě moc neukázal vlastní tvorbu, hodím sem aspoň část HTML/PUG struktury vrstevnatých blog postů, které vidíš na hlavní stránce blogu. S CSS si poraď sám.

PUG/HTML

nav.nav(role='navigation')
            ul.nav__list
              each tag, name in mvb.groupedArticles.byTag 
                br
                label.task__tag2.task__tag--copyright
                  input#group-1(type='checkbox' hidden='')
                  label.shrink.fa.fa-angle-right= name
                    span
                  ul.group-list  
                    each article in tag
                      br 
                      li
                        a.articlelink(href=`articles/${article.id}.html`)= article.title

Dělám si srandu. CSS přidám někdy později. V té době to byl ještě slušný chaos.

Měj se a zase někdy.
